# Level 1

### ✏️ 같은 숫자는 싫어

주어진 숫자 배열에서 연속적으로 반복되는 수 하나만 남겨 새로운 배열 반환

- 조건
  - 원소는 순서를 유지해야한다.

- 예시 
  - input: [3,3,2,3] 
  - output: [3,2,3]

> **풀이내용**
```java
function solution(arr) {
  let answer = [];
  for(let i = 0; i < arr.length; i++){
        if(arr[i] != arr[i+1]){
            answer.push(arr[i])
        }
    }
    return answer
}
```
